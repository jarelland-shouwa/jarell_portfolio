Virtual Photobooth

Product showcase
My team mate
Product showcase
Development collage

We were tasked to create a virtual photo booth to be used for a SST's anniversary event (hypothetically). We ideated many prop ideas and customisations. We included the ability to configure eye and head props, as well as the foreground and background. We used an existing pre-trained face shape predictor that uses a set of 68 landmarks to identify and locate key features of the user's face.

During the development, we encountered issues like ensuring the virtual prop stays on the user's face and remains there even if the face rotates (using trigonometry). We also had to code the two layers – the background and virtual prop – to work well together. Not only did I had fun creating my own suitable props and experimenting, but I also learnt how landmarks on a face work and to use OpenCV on Python.
