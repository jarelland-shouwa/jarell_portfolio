Fake news detection model

One of the data visualisations created using Matplotlib
Thought process when training the Teachable Machine model
How we chose our final model

My team trained a supervised machine learning model to detect fake news using linguistic features, using Jupyter Notebook. We targeted word frequencies and settled on logistic regression. We first pre-processed the data set used, e.g. removing stopwords like "as" and "being". We then created 3 data visualisations to better understand the data we were working with, e.g. word frequencies. After training two models, we used two evaluation metrics to evaluate them by calculating precision, recall and F1 score.
Our final model's results:
Recall = 99.6%
Classification Accuracy = 99.6%

When training the model, we encountered errors like convergence errors. Nonetheless, we were able to train the model using techniques like regularisation to properly fit our first model onto our data set (to avoid under and overfitting).
